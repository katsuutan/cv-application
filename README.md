# CV Application

A React-based CV/résumé builder where users can input and edit their general information, education, and work experience.

## Built With
- React
- Vite
- CSS

## Features
- Edit and submit toggles for each section
- Live preview of formatted CV information
- Date formatting for education and experience entries
